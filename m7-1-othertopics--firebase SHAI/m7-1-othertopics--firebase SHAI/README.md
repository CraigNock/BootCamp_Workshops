# Firebase with Google Auth

## Setup

### Installation

#### Backend

```bash
yarn install
```

#### Frontend

```bash
cd client
yarn install
```

### Start

This tutorial will require 2 terminals open. One to manage the FE (`/client`), and one for the BE.

#### Backend

```bash
yarn dev
```

#### Frontend

```bash
cd client
yarn start
```
